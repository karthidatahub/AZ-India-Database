# 🛒 AZ India Database

## 📌 Project Overview

The AZ India Database is a MySQL project developed to simulate an E-Commerce system.

This project manages customer details, product information, and customer orders. It also includes SQL business analysis queries to generate useful insights such as top customers, revenue analysis, product performance, and city-wise sales.

This project was designed and developed as part of my Data Analyst portfolio.
